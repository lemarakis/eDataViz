# eDataViz  
**Διαδραστική Οπτικοποίηση Γαλακτοπαραγωγικών Δεδομένων**

Το **eDataViz** είναι μια διαδραστική web εφαρμογή οπτικοποίησης δεδομένων,
η οποία παρουσιάζει στοιχεία γαλακτοπαραγωγής προβάτων με καθαρό και
κατανοητό τρόπο.

Στόχος της εφαρμογής είναι να βοηθήσει τους χρήστες (γεωπόνους, ερευνητές,
φοιτητές) να εντοπίζουν τάσεις, μοτίβα και διακυμάνσεις στην παραγωγή,
μέσα από διαδραστικά γραφήματα και φίλτρα.

---

## Βασικά Χαρακτηριστικά

- Οπτικοποίηση δεδομένων ανά:
  - Παραγωγικό έτος
  - Γαλακτική περίοδο
  - Μήνα
  - Κλάσεις γαλακτοπαραγωγής
- Διαδραστικά γραφήματα (line charts, bar charts, κατανομές)
- Φίλτρα για έτος, φυλή και άλλα κριτήρια
- Παρουσίαση γραφημάτων και πίνακες δεδομένων
